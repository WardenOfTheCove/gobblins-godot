# gobblins-godot
Gobblins is a simple platform parkour game with two levels. It is my first Godot project. No AI was used in this project
For installation, just download the .zip or the .tar.gz files in the releases. The character sprite isn't very polished, but will be cleaned up in future updates. __THIS PROJECT DID NOT USE AI__
